FaSteR-Spaceman. Capstone Project: FaSteR Spacemen, A smarter way to defrag XFS. The major enhancements that this project aims to add to the operation of xfs_fsr and the need for creation of xfs_spaceman constitute as our goals.

They are as follows:

  • Finding the owners of free-blocks in the filesystem.
  • Enhanced/faster filesystem scanning.
  • Ability to defragment only portions of files.
  • Defragmenting the free space.

    • Parts of the Project have been accepted to the Linux Kernel and in xfs-progs.


    Push Notification Using GCM(Google Cloud Messaging).

    I completed this Proof of Concept as a Summer Intern. The app was developed on Android platform.

    The app allows 3rd party application servers to send push notifications to android phones; also the app makes an entry about the event in the device’s calendar. To receive notifications, the Android handsets have to be registered with the GCM service as well as the 3rd party server. The data sent from the application servers is permutable.

    Virtual Learning Environment(VLE).

    This project was a complete virtual learning environment system which had key features like cryptographic hash for password storage, role based access control and identification of end user support for multiple media types and extensions. Discussion feeds were also added at the end of each video lecture or assignment for student discussions and comments. The project used servlets, JSP and MongoDB databases.


    School Admission Process Management System for Akshara International School.

    The project used Visual basic programming for the front end and Oracle databases (Oracle 10g) for the backend. ￼￼The challenge was to come up with an adaptive testing mechanism ￼that could effectively sort students without discouraging them. An analysis module was also added using Microsoft Chart Control 8.0 to depict the performance of students. The process ended with the generation of I-cards, which were embedded with QR barcodes. My contribution as an integral member of a team of four students was to design the algorithm for switching levels in the adaptive exam, the analysis module and the QR barcode generation.
